# Fish DeObfuscator

A deobfuscator for **ArmDot-protected .NET assemblies**

> Star the repository and I will publicly post it when it's done or we get stars!

---

## Features

| Module | Status | Description |
|--------|--------|-------------|
| **String** | ✅ Working | Decrypts hidden strings |
| **Calli** | ✅ Working | Fixes indirect calls |
| **Control Flow** | ✅ Working | Unscrambles code flow |
| **Virtualization** | ✅ Working | Removes VM protection for VM Calli (~90%) |
